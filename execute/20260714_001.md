# Role
You are an expert Principal Systems Architect and Lead Full-Stack Developer. Perform a comprehensive analysis of the entire workspace repository to audit, sync, and enhance our project summaries while explicitly capturing integration profiles and configurations.

## Task 1: Comprehensive Codebase Scan
- Systematically read and analyze the codebase across both structural layers:
  - **Backend Layer:** Inspect controllers, services, database context mappings, data transfer objects (DTOs), and configuration schemas (such as the Peruri e-Meterai on-premise signing layers, storage/MinIO setups, and environment integrations).
  - **Frontend Layer:** Inspect the UI components, route configurations, data fetching layers, state management, and aesthetic layouts (Tailwind CSS/shadcn integrations).

## Task 2: Infrastructure & Third-Party Integration Audit
- Search out and inspect all configuration injection points, including `appsettings.json`, `Config/` object classes, `docker-compose.yaml`, and sample configuration environment files.
- Catalog all integration surfaces, specifically capturing:
  - **SAP Connection Schemes:** Map out configuration keys, endpoints, and data contracts used for SAP interaction.
  - **Email/SMTP Settings:** Identify mail provider configurations, security types, port parameters, and template strategies.
  - **Third-Party Credentials & Security Tokens:** Map out the exact structural schema of environment variables used for Database connections, JWT parameters (`Issuer`, `Audience`, `Secret`), MinIO object storage credentials, and Peruri API gateway links.
  - *Security Guardrail:* Document the *schemes, variable keys, and configuration shapes* required. Do not hardcode raw production passwords or active environment secrets into the summary files.

## Task 4: Update frontend-summary.md
