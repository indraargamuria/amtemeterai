- **New Configuration Infrastructure:**

    | Column | Type | Constraints |

    | Value | string | Required |
    | Description | string | Nullable |
    | UpdatedAt | DateTime | Required |

  - Execute atomicity within an explicit database transaction:
    - Set `IsVoided = true`.
    - Update `Status = InvoiceStatus.Canceled`.

  - Update execution flags to explicitly handle invoicing state redirection, allowing the background processor to safely evaluate it again.
